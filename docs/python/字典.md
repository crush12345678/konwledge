1. Python中 ***dic.items(),dic.keys(),dic.values()***.

   * 助理解 dic={"age":1,"name":"guanguan","sex":"female"}

     `````python
     IN:dic.keys()
     OUT:['age','name','sex']
     `````

     ```python
     IN:dic.values()
     OUT:[1,'guanguan'，'female']
     ```

     ```python
     IN:dic.items()
     OUT:[('age',1),('name','guanguan'),('sex','female')]
     ```
