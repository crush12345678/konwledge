## \# include <stdlib.h>

### int *p = (int *)malloc(n * sizeof(int));

| int *       | 首地址转化为int *类型，赋值给指针变量p,p指向第一个整数的位置 |
| ----------- | ------------------------------------------------------------ |
| malloc ()   | 在内存的动态存储存储区种分配长度为zise的内存空间             |
| n           | n个int类型数据的内存块                                       |
| sizeof 运算 | sizeof运算符获取当前系统int类型的内存空间                    |

---

## 字符指针变量常用的两种方法

1. 是让其指向已经存在的字符数组的首地址
2. 是让其指向一个动态分配的内存块
